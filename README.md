# MyBudgetLocal (Web)

Petite application web responsive (phone/tablette/PC) : inscription/connexion locales, puis tableau calculateur de budget. Aucune base de données, tout est stocké dans `localStorage` (démo). Idéal pour un POC avant d'ajouter un backend.

## Scripts
- Ouvrir `index.html` dans votre navigateur, ou via un serveur statique (VS Code Live Server).
- Connexion → redirection vers `dashboard.html`.
